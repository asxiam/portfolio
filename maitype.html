MaiType

MaiType is a typography study of Japanese letterforms.

Project Tags

    Type

    Motion Design

"Mai" comes out to a single, yet complex character in Japanese, ruled by a completely different set of typography guides and foundations than Latin. It's alternative meaning is of celebration and dance. As a design challenge, I set out to create a set of characters, reskinned in a new way. Every day, for a month.
